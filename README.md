# 🔐 Firebase Authentication
A modern **React + Vite** authentication application built using **Firebase Authentication**. This project allows users to securely create an account, log in with their email and password, and access protected routes using React Router. Firebase Authentication provides secure user management with minimal setup.
## 🚀 Live Demo

🌐 **Netlify:** https://firebasedk.netlify.app/

## ✨ Features
* 🔑 User Registration (Sign Up)
* 🔓 User Login
* 🚪 User Logout
* 🛡️ Protected Routes
* 🔥 Firebase Authentication
* ⚡ Built with React + Vite

## ⚙️ Installation
Clone the repository:
```bash
git clone https://github.com/ohitzdhanush/<repository-name>.git
```
Move into the project directory:
```bash
cd <repository-name>
```
Install dependencies:
```bash
npm install
```
Start the development server:
```bash
npm run dev
```
Build for production:
```bash
npm run build
```
---
## 🔥 Firebase Setup

1. Create a Firebase project.
2. Enable **Email/Password Authentication** in the Firebase Console.
3. Copy your Firebase configuration into `src/Firebase.js`.
4. Run the application.

Firebase Authentication supports secure email/password authentication and integrates easily with React applications.
